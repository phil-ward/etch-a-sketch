# Etch-a-Sketch

This is my implementation of The Odin Project's Etch-a-Sketch project from the Foundations Course.

## Project Description

A browser-based sketchpad that allows users to draw by hovering over a grid of squares. The grid dynamically changes color as the mouse passes over it, creating a pixelated drawing effect similar to an Etch-a-Sketch toy.

## Skills Demonstrated

- DOM manipulation with JavaScript
- Creating and removing HTML elements dynamically
- Event listeners (hover effects, button clicks)
- Flexbox layout for grid creation
- User input handling with prompts
- Dynamic styling with JavaScript
- Problem-solving and debugging

## Features

- 16x16 grid that initializes on page load
- Hover effect that changes square colors to grey
- Button to customize grid size (up to 100x100)
- Grid regenerates while maintaining the same total canvas size (960px)

## How to Use

1. Hover your mouse over the grid to start drawing
2. Click "Change Square Dimensions" to create a new grid
3. Enter a number (less than 100) for the grid size
4. Start drawing on your new canvas!
